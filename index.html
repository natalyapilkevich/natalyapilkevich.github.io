Во всех проектах применяются следующие технологии и методы:

  • Использован препроцессор SASS

  • Кроссбраузерность и кроссплатформенность

Проекты

  1. Jewellery - ювелирные украшения

    Интернет-магазин ювелирных украшений.

    Проект представлен на 3-х страницах (главная, страница каталога и карточка продукта)

    Для главной страницы реализованы всплывающее окно с формой по клику на ссылку "Login", слайдер, раскрывающийся список (аккордеон).

    Для страницы каталога (переход по "Shop now" на главной) реализованы раскрывающийся список (аккордеон) для фильтров, сброс выбранных фильтров по кнопке "Clear All", на мобильных устройствах показ секции фильтров во всплывающем окне.

    Для страницы карточка продукта реализован слайдер на мобильных устройствах для фотографий, всплывающее окно по кнопке "Add to cart".

    Для мобильных устройств предусмотрено раскрывающееся меню.

    А здесь можно посмотреть рабочий репозиторий этого проекта на GitHub.

  2. Израиль

    Посадочная страница компании, специализирующейся на учебных программах и стажировках в Израиле.

    Одностраничный сайт на флексбоксах.

    По кнопке "Заказать звонок" реализовано всплывающее окно с формой. Для формы предусмотрена валидация и показ ошибки при невалидных данных.

    Раздел "Программы" представляет собой табы, он и фотогаллерея "Жизнь в Израиле" на мобильных устройствах преобразовывается в слайдер.

    Раздел "Частые вопросы" представляет собой аккордеон, а раздел "Отзывы" - слайдер.

    А здесь можно посмотреть рабочий репозиторий этого проекта на GitHub.

    А здесь можно посмотреть рабочий репозиторий этого проекта на GitHub.

    Сервис просмотра изображений.
